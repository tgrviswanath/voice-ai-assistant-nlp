Sample files:
  text_questions.txt         - 15 questions to type into the text input
  knowledge_base_entries.txt - 4 KB entries to add for better answers

Usage:
  Text input: type any question from text_questions.txt and press Send
  Voice input: record yourself asking any of the questions
  KB: add entries from knowledge_base_entries.txt to improve answers

Note: Audio samples cannot be generated programmatically without a TTS engine.
To test voice input, record yourself asking any question from text_questions.txt
using your microphone (WAV or MP3 format, 16kHz recommended for Whisper).
